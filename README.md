# 张鹏翔 · 创始人简介 deck

智械工坊创始人 **张鹏翔（Pengxiang Zhang）** 的独立简介演示 —— 点击推进的 16:9「伪视频」网页 deck（Vite + React + TS），暖琥珀电影感，与 [mech-claw 路演站](https://github.com/longlivethemech/mech-claw-roadshow) 同一套设计语言。

## 内容（2 章 8 页）

1. 封面 · 张鹏翔
2. 信念 · 物理世界 × 数字世界的交汇
3. 学术轨迹 · 人大附中 → 清华 → MIT → 阿贡国家实验室
4. 学术影响力 · 1502 引用 / h-index 14 / i10 18
5. 代表作 · Science / Nature 子刊 / PRL
6. 转身 · 放下学术轨道，回国创办智械工坊
7. 洞察 · LLM 与具身机器人之间的「缝隙」
8. 核心理念 · 让机械生命，和人类一起成长

履历数据来源：[Google Scholar](https://scholar.google.com/citations?user=OoPvVZgAAAAJ) · [ORCID](https://orcid.org/0000-0003-4171-9866)。

## 本地运行

```bash
npm install
npm run dev      # http://localhost:5188
npm run build    # 产物在 dist/
```

肖像照：把照片命名为 `zhang-portrait.jpg` 放入 `public/assets/`，封面会自动显示。

## 部署

`main` 分支推送后由 GitHub Actions 自动构建并发布到 GitHub Pages。
